Proyecto Final – Diplomado en Desarrollo Web Full Stack

Este proyecto es el resultado final del diplomado, integrando todos los conocimientos aprendidos a lo largo de los módulos.
El desarrollo combina un frontend en Angular con un backend en Node.js, Express y MongoDB, creando una aplicación web funcional, escalable y conectada de forma completa.


Estructura del Proyecto

Frontend: construido con Angular, incluye componentes de inicio, registro, login y dashboard.

Backend: desarrollado con Node.js y Express, conectado a MongoDB para la gestión de usuarios.

Base de datos: MongoDB con modelo de usuario (nombre, correo, contraseña y rol).

Estructura de carpetas:

proyecto-final-fullstack/
│
├── frontend/      → aplicación Angular (interfaz)
├── backend/       → servidor Node.js + Express
└── README.md      → documentación general del proyecto

Tecnologías Utilizadas

Frontend: Angular, HTML, CSS, TypeScript, Bootstrap

Backend: Node.js, Express.js, Mongoose, Dotenv, CORS

Base de datos: MongoDB

 Ejecución del Proyecto
1️Backend

Desde la carpeta /backend:

npm install
npm start


Servidor disponible en: http://localhost:3000

Frontend

Desde la carpeta /frontend:

npm install
ng serve


Aplicación disponible en: http://localhost:4200

Funcionalidades Principales

Registro y autenticación de usuarios.

Validaciones en formularios (correo, contraseña, campos requeridos).

Comunicación entre Angular y el servidor a través de HTTP.

Visualización dinámica de datos con data binding.

Manejo de sesiones con localStorage.

Redirección según autenticación del usuario.

 Aprendizajes Clave

Durante este proyecto final se reforzaron conceptos esenciales del desarrollo web full stack:

Conexión entre frontend y backend.

Manejo de APIs RESTful.

Validación de formularios y autenticación.

Organización profesional de proyectos con Angular y Node.

Control de versiones con Git y GitHub.
